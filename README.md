# 🛠 Installation

Clone the repository

```bash

```

Go inside the folder

```bash
cd Matplotlib
```

Install Matplotlib

```bash
pip install matplotlib
```

Or

```bash
pip install matplotlib numpy pandas
```

# 📌 Basic Example

```python
import matplotlib.pyplot as plt

x = [1,2,3,4,5]
y = [2,4,6,8,10]

plt.plot(x, y)

plt.title("Simple Line Plot")
plt.xlabel("X-axis")
plt.ylabel("Y-axis")

plt.grid(True)

plt.show()
```
